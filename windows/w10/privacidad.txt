= Windows 10 privacidad
para ir a configuración, pulsar <kbd>Windows+X</kbd>
== configuración -> Privacidad
  * General: todo desactivado
  * Voz: desactivado
  * comentarios y diagnósticos:
    * Marcar **Datos de diagnósticos necesarios**
    * Desactivar todo
    * Eliminar
    * Frecuencia de comentarios: **nunca**
  * Historial de actividad:
    * desactivar todo
    * borrar
    * si la cuenta está vinculada a una cuenta de Microsoft, a través de [[https://account.microsoft.com/privacy/|este enlace]] se puede gestionar
  * Ubicación
  * Cámara
  * Micrófono

== services.msc
  * ejecutar <kbd>windows+R</kbd>: **services.msc**
  * buscar **Diagnostic Execution Service**
  * tipo de inicio: **Deshabilitado**
